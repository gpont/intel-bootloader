# Intel bootloader ![Version](https://img.shields.io/badge/version-1.0.0-lightgrey.svg)

Written just for fun in the unknown year minimal Intel x86 bootloader.

## Usage

### Requirements

* GCC
* LD
* NASM
* QEMU (for running)

### Compiling

```bash
make
```

### Running (in qemu)

```bash
make run
```
